Smart Doorway Reminder System

Technologies Used: Wokwi Online Simulation

Problem Statement

In our fast-paced daily routines, people frequently forget essential items like keys, wallets, ID cards, or medications when leaving their homes. This forgetfulness leads to wasted time, unnecessary stress, and reduced productivity as individuals are forced to return home to retrieve these items.

Existing Factors

Currently, most people rely on manual systems like leaving sticky notes on the door, setting phone alarms, or just memory to ensure they have everything. These methods are prone to failure because they lack context-awarenessâ€”an alarm might go off when you aren't leaving, or a note might be ignored due to habituation.

Solution

The Smart Doorway Reminder System uses an IoT-based approach to passively monitor whether essential items are picked up before a person exits the house. Simulated using Wokwi, the system detects motion near the doorway and cross-references it with sensor data from item storage locations. If a user approaches the door without all required items, the system triggers an immediate visual and auditory alert, ensuring nothing is left behind.
